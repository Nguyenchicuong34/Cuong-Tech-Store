#  Cuong Tech Store

###  Overview
**Cuong Tech Store** is a full-featured e-commerce platform designed to provide a seamless shopping experience for technology enthusiasts. This project demonstrates my ability to build a modern web interface combined with functional backend logic.

One of the highlights of this project is the **interactive UI**, including a custom **Christmas-themed mode** with falling snow effects to enhance user engagement during the holiday season.

---

###  Key Features
* ** Product Management:** Browse, search, and filter technology products (Phones, Laptops, Accessories).
* ** Shopping Cart System:** Add to cart, update quantities, and calculate total price dynamically.
* ** Responsive Design:** Fully optimized for Desktop, Tablet, and Mobile devices.
* ** Interactive UI:** Special Christmas theme with animated snow effects and 3D elements.
